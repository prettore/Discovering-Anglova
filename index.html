Overview

This website showcases the data analyses conducted to explore the characteristics of the Anglova Scenario Vignette 2. The analyses focus on spatiotemporal mobility metrics and communication patterns within the scenario.

About Anglova Scenario

The Anglova Scenario was developed as part of the NATO IST-124 Research Task Group on Heterogeneous Networks: Improving Connectivity and Network Efficiency. It is an emulation scenario specifically developed for experimentation over emulated tactical networking environments.

Vignette 2 focuses on Troop Deployment and consists of detailed movement and pathloss parameters for a battalion-sized deployment.

About This Project

This website was created to showcase the data analyses conducted on the Anglova Scenario Vignette 2, focusing on spatiotemporal mobility metrics and communication patterns.

Data Source

The analyses presented on this website are based on the Anglova Scenario Vignette 2, which was developed as part of the NATO IST-124 Research Task Group.

The original data can be accessed from the Anglova website.

Analysis Methodology

The trace file was characterized by exploring the spatiotemporal mobility metrics, including speed, distance, acceleration, and path characteristics.

Communication patterns were analyzed across different technologies: Bluetooth, UHF, VHF, and Wi-Fi.
